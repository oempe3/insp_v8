# Inspeção On-Line – Termelétrica Pernambuco III

Este projeto contém um site estático para publicação no GitHub Pages com dois formulários:
- Formulário **INTERNO** (`interno.html`)
- Formulário **EXTERNO** (`externo.html`)

## Como usar

1. Suba todo o conteúdo da pasta `inspecao-online/` para o repositório no GitHub.
2. Publique via GitHub Pages (`Settings → Pages → Branch: main`).
3. No arquivo `interno.html` e `externo.html`, substitua o valor `action="https://script.google.com/macros/s/SEU_SCRIPT_ID/exec"` pelo ID real do seu Google Apps Script WebApp.
4. Ajuste o logo `logo.png` e as cores em `style.css` conforme sua identidade visual.
5. Verifique responsividade nos dispositivos móveis.
6. Depois prossiga com configuração do Apps Script para gravação na planilha, geração de PDF, envio de email e upload de foto.
